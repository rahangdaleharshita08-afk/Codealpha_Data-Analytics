# CodeAlpha Data Analytics Internship

This repository contains the projects completed during the CodeAlpha Data Analytics Internship.

## ✅ Task 1: Web Scraping
Extracted book data from a website using BeautifulSoup and created custom datasets.

## ✅ Task 2: Exploratory Data Analysis (EDA)
Explored the dataset to identify patterns, trends, and data quality issues.

## ✅ Task 3: Data Visualization
Created charts and graphs to transform raw data into meaningful insights.

## ✅ Task 4: Sentiment Analysis
Performed sentiment analysis on book data by classifying ratings into Positive, Neutral, and Negative categories and visualizing sentiment patterns.

## Tools Used
- Python
- Pandas
- BeautifulSoup
- Matplotlib
- Google Colab
